### 🛒 Dark-Shops

> *A fairer economy that respects your time as a prospector.*

A balance mod focused on making in-game shops and exchange systems more rewarding.
Vanilla shop prices and trade quantities often feel unrewarding — this mod adjusts costs and quantities across all shop categories so that trading feels like a viable part of your progression rather than something to ignore.

**Shops affected:** Food Shop — Crops Exchange — Fish Exchange — Biomass Exchange — Vestige Exchange

No new items or mechanics — just a cleaner, fairer economy.

## 🔧 Installation

### ▶ Client (Local Machine)

1. Shut down Icarus completely
2. Place the mod file (`.pak` or `.EXMODZ`) in:
```
(drive):\Program Files (x86)\Steam\steamapps\common\Icarus\Icarus\Content\Paks\mods
```
3. If the `mods` folder doesn't exist, create it
4. In multiplayer, **all players must use the exact same mods** to avoid issues

> 💡 **Recommended:** Use **[IMM — Icarus Mod Manager](https://github.com/jimk72/IcarusModManager)** by JimK72 for easier installation and conflict detection.

### 🖥️ Dedicated Server

1. Shut down the server completely
2. Place the mod file in:
```
(drive):\<IcarusServerFolder>\Icarus\Content\Paks\mods
```
3. If the `mods` folder doesn't exist, create it
4. **Server and all clients must use the exact same mods**
